# Assignment 3 – Study Notes & Quiz Android App

## Overview
This Android application allows users to create and manage study notes, attempt quizzes, and track quiz scores. The app preserves user state during screen rotation, background/foreground transitions, and app restarts.

## Features
- Add study notes
- Save notes using SQLite database
- View saved notes
- Quiz with multiple-choice questions
- Quiz score calculation
- Quiz score persistence using SharedPreferences
- State preservation on rotation
- Offline functionality

## Navigation Flow
Main Screen →
- Add Note → Save Note
- View Notes → Display Saved Notes
- Start Quiz → Submit Quiz → View Score

## Technologies Used
- Java
- XML
- SQLite
- SharedPreferences
- Android Studio

## Testing
- Tested on Android Emulator
- Tested on Physical Android Device
